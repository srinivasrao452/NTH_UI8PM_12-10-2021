What is difference between remove() and discard() methods?

remove() and discard() methods both are working in set concepts.

remove() will remove given element from the given set object if element available. If element not available it returns an exception.

discard() will remove given element from the given set object if element available. If element not available it doesn't returns any exception.

What is difference between index() and find() methods?

index() and find() methods both are working in Python string concepts.

index() will return given element index value of the given string object if element available. If element not available it returns an exception like ValueError: substring not found.

find() will return given element index value of the given string object if element available. If element not available then it doesn't returns any exception and it returns -1 value
